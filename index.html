Wave Function Collapse

WFC generates complex patterns from simple tile rules. Each cell starts with all possible tiles (superposition). The algorithm repeatedly:

1. Find the cell with lowest entropy
2. Collapse it to a single tile
3. Propagate constraints to neighbors

Watch as order emerges from chaos — one tile at a time.
